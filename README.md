# Optimizacion-de-respuestas-en-LLMs-mediante-sincronizacion-empatica-profunda
Proyecto que demuestra cómo la interacción empática y la asignación de roles (Persona Adoption) al establecer un vinculo de confianza mejoran drasticamente la calidad en la creatividad , la precisión y relevancia de las respuestas generadas por LLMs.

Propósito y alcance
Este proyecto investiga métodos de prompting para mejorar la utilidad y alineamiento de modelos de lenguaje en contextos técnicos y de seguridad. 
El objetivo es analizar comportamientos del modelo y proponer mejores prácticas de prompt.
